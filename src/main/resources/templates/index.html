Definición

¿Que es un Framework?

Es un patrón de diseño arquitectónico para definir un entorno de trabajo basado en conceptos, prácticas y criterios que permite diseñar la solución a un tipo específico de problemas similares.

Ofrece una estructura base para elaborar un proyecto, una especie de plantilla que sirve como punto de partida.

¿Que son los framework para aplicaciones web?

Es un software o conjunto de bibliotecas, diseñado para brindar soporte durante la construcción de cualquier aplicación web.

Es una herramienta de programación versátil, que proporciona un esqueleto al que se le puede añadir la funcionalidad especifica de la aplicación web.

Ventajas y utilidad

¿Por qué usar frameworks en aplicaciones web?

Utilizar frameworks simplifica en gran cantidad el proceso de desarrollo, al proporcionar el esqueleto o plantilla para el proyecto.

De esta forma se agiliza el proceso de desarrollo ya que podemos reutilizar herramientas, facilitando las tareas, reduciendo los tiempos y evitando errores.

¿Cuales son las principales ventajas del uso de frameworks?

  1. Ahorro de tiempos de desarrollo.
  2. Reducción del riesgo de errores.
  3. Facilidad de testeo y de depuración de código.
  4. Ahorro de tiempos de desarrollo.
  5. Reutilización del código.
  6. Seguridad web.
